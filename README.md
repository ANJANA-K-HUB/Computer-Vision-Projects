# 🤖 Computer Vision & Machine Learning Projects

A collection of Python-based computer vision and machine learning projects built using OpenCV, scikit-learn, and scikit-image.

---

## 📂 Projects

### 1. 🚗 Number Plate Detection & Blurring

Detects vehicle license plates in images using OpenCV's Haar Cascade classifier and optionally blurs them for privacy.

**Highlights:**
- Real-time plate detection with bounding boxes
- Privacy anonymization via median blurring
- Lightweight — no model training required

**Tech:** `OpenCV` · `NumPy` · `Matplotlib`

📁 [`Number_Plate_Detection_Project.ipynb`](./Number_Plate_Detection_Project.ipynb)

---

### 2. 🛍️ Image Classification

Classifies images into four categories (Flowers, Butterfly, T-Shirt, TV) using a Support Vector Machine (SVM).

**Highlights:**
- Multi-class image classification (4 categories)
- Full ML pipeline: load → preprocess → train → evaluate
- Evaluation with accuracy score, confusion matrix & classification report

**Tech:** `scikit-learn` · `scikit-image` · `NumPy` · `Matplotlib` · `Seaborn`

📁 [`Image_Classification_Project.ipynb`](./Image_Classification_Project.ipynb)
