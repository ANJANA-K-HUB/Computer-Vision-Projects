# Image Classification Using Support Vector Machine (SVM)

This repository contains an **Image Classification** project developed using Python and Jupyter Notebook, where a **Support Vector Machine (SVM)** model is used to classify images into predefined categories. The project demonstrates how traditional machine learning techniques can be applied to image-based problems through effective feature extraction and preprocessing.

## Project Overview

Image classification involves assigning a label to an image based on its visual characteristics. In this project, images are processed and transformed into feature vectors, which are then used to train an SVM classifier. The trained model learns decision boundaries that separate different image classes with maximum margin.

This project highlights the use of SVM as an efficient and reliable machine learning algorithm for image classification tasks, especially when working with limited datasets.

## Objectives

- Understand the application of Support Vector Machine (SVM) in image classification  
- Perform image preprocessing and feature extraction  
- Train an SVM model for multi-class image classification  
- Evaluate model performance using standard machine learning metrics  
- Visualize training results and predictions
